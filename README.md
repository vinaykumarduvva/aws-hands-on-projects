- ✅ Free Tier compatibility status
- 💵 Best-case and worst-case cost estimates

## 🔖 Navigating This Repository

Each project folder is a self-contained learning module. Dive into any directory to find:

1. **`README.md`** — The master setup guide (Console + CLI).
2. **`images/`** — Visual proof of a working setup.
3. **`scripts/`** — Reusable Bash and PowerShell automation.
4. **`docs/`** — Architecture diagrams, deep-dive notes, and troubleshooting matrices.

*Recommendation: Follow along chronologically if you are a beginner, or jump straight to the intermediate modules if you have the prerequisites!*
